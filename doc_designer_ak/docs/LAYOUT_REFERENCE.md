# AK Document Designer — Layout & CSS Reference

This document covers all built-in CSS classes available for document templates. Use these classes for consistent, responsive styling across all templates.

---

## Document Container

### `.ak-document`
Main container for the document content. Provides white background, border radius, shadow, and padding.

```html
<div class="ak-document">
  <!-- All template content goes here -->
</div>
```

**Styles:** `background: #fff`, `border-radius: 8px`, `box-shadow`, `padding: 40px`, `margin-bottom: 24px`

### `.ak-document-form`
Alternative container used for the form wrapper on the shared page. Same visual style as `.ak-document`.

### `.ak-document-body`
Wrapper for the main template body content (used internally by the shared page).

---

## Layout Sections

### `.ak-header`
Document header area. Has a bottom border for visual separation.

```html
<div class="ak-header">
  <h1>Invoice {{ doc.name }}</h1>
  <p>Date: {{ doc.posting_date }}</p>
</div>
```

**Styles:** `margin-bottom: 24px`, `padding-bottom: 16px`, `border-bottom: 2px solid #e5e7eb`

### `.ak-footer`
Document footer area. Has a top border.

```html
<div class="ak-footer">
  <p>Thank you for your business.</p>
  <p>{{ doc.company }} — {{ doc.company_address }}</p>
</div>
```

**Styles:** `margin-top: 24px`, `padding-top: 16px`, `border-top: 1px solid #e5e7eb`, `font-size: 12px`, `color: #6b7280`

### `.ak-section`
General content section with bottom margin.

```html
<div class="ak-section">
  <h3>Bill To</h3>
  <p>{{ doc.customer_name }}</p>
</div>

<div class="ak-section">
  <h3>Ship To</h3>
  <p>{{ doc.shipping_address }}</p>
</div>
```

**Styles:** `margin-bottom: 24px`

### `.ak-cover-page`
Cover page with centered text and generous padding. Typically used for the first page of proposals or contracts.

```html
<div class="ak-cover-page">
  <h1>PROPOSAL</h1>
  <p>Prepared for {{ doc.customer_name }}</p>
  <p>{{ nowdate() }}</p>
</div>
```

**Styles:** `text-align: center`, `padding: 60px 40px`, `margin-bottom: 32px`

### `.ak-page-break`
Forces a page break when printing or generating PDF. No visual effect on screen.

```html
<div class="ak-page-break"></div>
```

**Styles:** `page-break-after: always`, `height: 0`, `margin: 0`

## Two-Column Layout

Use an HTML table with the `.ak-two-col` class for side-by-side content:

```html
<table class="ak-two-col">
  <tr>
    <td>
      <h4>Bill To</h4>
      <p>{{ doc.customer_name }}</p>
      <p>{{ doc.customer_address }}</p>
    </td>
    <td>
      <h4>Ship To</h4>
      <p>{{ doc.shipping_address_name }}</p>
      <p>{{ doc.shipping_address }}</p>
    </td>
  </tr>
</table>
```

### Classes
| Class | Purpose |
|---|---|
| `.ak-two-col` | Table container, 100% width |
| `.ak-two-col td` | Each column: 50% width, top-aligned, padded |
| `.ak-col-left` | Left column with right padding |
| `.ak-col-right` | Right column with left padding |

### Responsive
On screens < 640px, columns stack vertically (each `td` becomes `display: block; width: 100%`).

## Notice Box

### `.ak-notice`
Highlighted notice/warning box with amber styling.

```html
<div class="ak-notice">
  This is a draft document and may change before final approval.
</div>

{% if doc.grand_total > 50000 %}
<div class="ak-notice">
  Orders over $50,000 require additional authorization.
</div>
{% endif %}
```

**Styles:** `background: #fef3c7`, `border: 1px solid #f59e0b`, `border-radius: 6px`, `padding: 12px 16px`, `color: #92400e`

#### Custom Notice Styles
Override colors with inline styles for different notice types:

```html
<!-- Error/danger notice -->
<div class="ak-notice" style="background: #fef2f2; border-color: #ef4444; color: #dc2626;">
  Payment is overdue!
</div>

<!-- Success/info notice -->
<div class="ak-notice" style="background: #f0fdf4; border-color: #22c55e; color: #16a34a;">
  Payment received. Thank you!
</div>

<!-- Blue info notice -->
<div class="ak-notice" style="background: #eff6ff; border-color: #3b82f6; color: #1d4ed8;">
  Please review all items carefully before approving.
</div>
```

## Interactive Fields

These classes are auto-generated by the helper functions. You generally don't write them by hand, but knowing them helps with custom CSS:

### `.ak-field`
Wrapper for every interactive field. Contains the label and input.

**Styles:** `margin-bottom: 16px`

### `.ak-field-label`
Label element above the input.

**Styles:** `display: block`, `font-size: 13px`, `font-weight: 600`, `color: #374151`, `margin-bottom: 4px`

### `.ak-field-mandatory`
Applied to `.ak-field` wrapper when the field is required. Adds a red asterisk after the label.

**CSS:** `.ak-field-mandatory .ak-field-label::after { content: " *"; color: #ef4444; }`

### `.ak-field-input`
The actual input/textarea/select element.

**Styles:** `width: 100%`, `padding: 8px 12px`, `border: 1px solid #d1d5db`, `border-radius: 6px`, `font-size: 14px`

**Focus:** `border-color: #4f46e5`, blue ring shadow

**Disabled:** `background: #f9fafb`, `color: #6b7280`

### `.ak-field-readonly`
Applied to `.ak-field` wrapper when the field is not editable. Gives the input a gray background.

### `.ak-field-error`
Applied to `.ak-field` wrapper when validation fails. Red border and shadow on the input.

## Form Table

### `.ak-field-table`
Table that contains auto-generated field rows from `ak_field_table()`.

```html
<table class="ak-field-table">
  <tbody>
    <tr><td><!-- field 1 --></td></tr>
    <tr><td><!-- field 2 --></td></tr>
  </tbody>
</table>
```

**Styles:** `width: 100%`, `border-collapse: collapse`, `margin: 16px 0`

### `.ak-field-table td`
**Styles:** `padding: 4px 8px`, `vertical-align: top`

---

## Items/Pricing Table

### `.ak-items-table`
Full-width table for line items. Has styled headers, hover effects, and number alignment.

```html
{{ ak_items_table(doc) }}
```

### Sub-classes
| Class | Purpose | Alignment |
|---|---|---|
| `.ak-items-table thead` | Header row background | — |
| `.ak-items-table th` | Header cell: uppercase, 12px, bold | Left (default) |
| `.ak-items-table td` | Data cell: 14px, bottom border | Left (default) |
| `.ak-items-col-num` | Row number column | Center, 40px |
| `.ak-items-col-qty` | Quantity column | Right |
| `.ak-items-col-rate` | Rate column | Right |
| `.ak-items-col-amount` | Amount column | Right |
| `.ak-items-col-net_amount` | Net amount column | Right |
| `.ak-items-total td` | Total row: larger text, top border | Right |

### Hover
`.ak-items-table tbody tr:hover` → `background: #f9fafb`

## Action Buttons

### `.ak-actions`
Container for action buttons. Centered with gap.

**Styles:** `display: flex`, `gap: 12px`, `margin-top: 32px`, `padding-top: 24px`, `border-top: 1px solid #e5e7eb`, `justify-content: center`

### `.ak-accept-btn` / `.ak-submit-btn`
Primary action button (indigo).

**Styles:** `padding: 12px 32px`, `background: #4f46e5`, `color: #fff`, `border-radius: 8px`, `font-size: 15px`, `font-weight: 600`

**Hover:** `background: #4338ca`

### `.ak-decline-btn`
Secondary/danger action button (red outline).

**Styles:** `padding: 12px 32px`, `background: #fff`, `color: #dc2626`, `border: 2px solid #dc2626`, `border-radius: 8px`

**Hover:** `background: #fef2f2`

## Responsive Behavior

At screen width < 640px:
- `.ak-document` / `.ak-document-form` → `padding: 20px`, `border-radius: 0`
- `.ak-shared-page` → `padding: 0`
- `.ak-two-col td` → `display: block; width: 100%` (columns stack)
- `.ak-actions` → `flex-direction: column` (buttons stack)

---

## Print / PDF Behavior

When printing or generating PDF:
- Body background set to white
- `.ak-shared-branding`, `.ak-shared-footer`, `.ak-actions`, `.ak-toast` → hidden
- `.ak-document`, `.ak-document-form` → no box-shadow, no padding

### Page Settings (on AK Document Template)
| Setting | Default | Description |
|---|---|---|
| Page Format | A4 | A4, Letter, Legal |
| Top Margin | 15mm | |
| Bottom Margin | 15mm | |
| Left Margin | 15mm | |
| Right Margin | 15mm | |

## Custom CSS

Add custom styles via the **Custom CSS** field on the template. These styles are injected after the default stylesheet.

```css
/* Override header style */
.ak-header {
    border-bottom-color: #4f46e5;
    border-bottom-width: 3px;
}

/* Custom notice color */
.my-custom-notice {
    background: #dbeafe;
    border: 1px solid #3b82f6;
    border-radius: 6px;
    padding: 12px 16px;
    color: #1e40af;
}

/* Make the items table header dark */
.ak-items-table thead {
    background: #1f2937;
}
.ak-items-table th {
    color: #fff;
}
```
