# boon — Unified Brand Style Guide

**Version 1.0 · February 2026**  
Wellness-Oriented Water Purification

---

## 01 · Brand Overview

boon is a wellness-native water purification brand. Bold, direct, unapologetic. We don't whisper about clean water. We make it unmissable.

### Positioning

The first water purification brand that looks like it belongs in wellness, not under a kitchen sink. boon sits at the intersection of premium wellness (AG1, Harley Meds) and functional water purification — rejecting the generic blue-and-white appliance aesthetic in favor of bold, design-led branding that treats water as a daily wellness ritual, not a commodity.

### Brand Personality

| Attribute | Description |
|-----------|-------------|
| Tone | Confident, direct, warm — never preachy or clinical |
| Energy | Bold but grounded — Liquid Death's edge meets AG1's substance |
| Voice | Short sentences. Plain language. Science without jargon. |
| POV | We lead with what boon adds to your life, not what it removes from water |

### Core Taglines

**Primary:** water, elevated.

**Secondary options:** "Your water is lying to you." · "Every drop matters." · "Purified. Mineralized. Ritual-ready."

## 02 · Color Palette — Crimson & Charcoal

A five-color system built on maximum contrast. **Light backgrounds (Cream) are the default.** Crimson is the hero. Void Black anchors dark moments. Warm Gray handles secondary information.

### Primary Colors

| Name | Hex | RGB | Role |
|------|-----|-----|------|
| **Cream** | `#F2EEEA` | 242, 238, 234 | **PRIMARY BACKGROUND** — default canvas for web, print, editorial |
| **Charcoal** | `#1A1A1A` | 26, 26, 26 | Primary text on light backgrounds |
| **Crimson** | `#C8302B` | 200, 48, 43 | Brand primary — wordmark, CTAs, key accents |
| **Void Black** | `#0A0A0A` | 10, 10, 10 | Dark-mode bg, packaging, hero sections |
| **Warm Gray** | `#8A8178` | 138, 129, 120 | Secondary text, captions, labels |

### Extended Palette

| Name | Hex | Usage |
|------|-----|-------|
| Light Border | `#D8D2CA` | Borders and dividers on light backgrounds |
| Crimson Hover | `#A52722` | Interactive hover states |
| Crimson Tint | `rgba(200,48,43,0.08)` | Tags, tinted backgrounds on light mode |
| Mid Gray | `#555555` | Tertiary text, disabled states |
| Dark Border | `#2A2A2A` | Borders on dark backgrounds |

### Color Ratios

**Light mode (default):** Cream 55% · Charcoal text 20% · Crimson 10% · Warm Gray 10% · Void Black 5%

**Dark mode (packaging, heroes, social):** Void Black 50% · Charcoal 25% · Crimson 10% · Cream 10% · Warm Gray 5%

Crimson is always used sparingly — it's a scalpel, not a paintbrush.

### Mode Usage

- **Light mode (default):** Web, editorial, product pages, print collateral. Cream `#F2EEEA` background, Charcoal `#1A1A1A` text, Crimson `#C8302B` wordmark and accents, Light Border `#D8D2CA` for dividers.
- **Dark mode:** Packaging, hero sections, social media, statement moments. Void Black `#0A0A0A` background, Cream `#F2EEEA` text, Crimson `#C8302B` wordmark and accents.

### Color Rules

**DO:**

- Use Cream `#F2EEEA` as the default background — not pure white
- Use Charcoal `#1A1A1A` for primary text on light backgrounds — not pure black
- Use Crimson for the wordmark, primary CTAs, and key data points only
- Use Warm Gray for secondary/supporting text and labels
- Reserve dark mode for packaging, hero banners, and social content

**DON'T:**

- Place Crimson text below 14px — contrast is insufficient at small sizes
- Use Crimson as a background fill (except hero CTA buttons or rare statement moments)
- Add blues, greens, or any colors outside this system
- Use pure black `#000000` or pure white `#FFFFFF` — always use Charcoal/Void Black and Cream

## 03 · Typography

A three-tier type system: Space Grotesk for display and headlines, DM Sans for body, Space Mono for technical data. All lowercase wordmark. Generous letterspacing on display.

### Wordmark

| Property | Value |
|----------|-------|
| Font | Space Grotesk Bold (700) |
| Case | Always lowercase — never "Boon" or "BOON" |
| Letterspacing | +0.15em (scaled proportionally) |
| Color | Crimson `#C8302B` on both light and dark backgrounds |
| Minimum size | 24px digital / 18pt print |

### Display Font — Space Grotesk

Source: Google Fonts · Variable font available · Weights: 300 400 500 600 700

| Use | Weight | Size | Letterspacing |
|-----|--------|------|---------------|
| Hero headlines | 700 (Bold) | 48–72px | +0.02em |
| Section headlines | 700 (Bold) | 28–36px | +0.01em |
| Sub-headlines | 500 (Medium) | 20–24px | 0 |
| Navigation / UI labels | 600 (Semi) | 14–16px | +0.02em |

### Body Font — DM Sans

Source: Google Fonts · Weights: 300 400 500 600 700

| Use | Weight | Size | Line Height |
|-----|--------|------|-------------|
| Body text | 400 (Regular) | 15–16px | 1.7 |
| Small body / captions | 400 (Regular) | 13–14px | 1.6 |
| Bold emphasis | 600 (Semi) | Inherit | Inherit |
| Buttons | 500 (Medium) | 14–16px | 1 |

### Mono Font — Space Mono

Source: Google Fonts · Weights: 400 700

| Use | Weight | Size | Notes |
|-----|--------|------|-------|
| Data points / specs | 400 | 12–14px | Crimson color for key stats |
| Labels / tags | 400 | 9–10px | All-caps, +2px letterspacing |
| Technical footnotes | 400 | 10–11px | Warm Gray color |

### Typography Rules

**DO:**

- Always write "boon" in lowercase — in all contexts, sizes, media
- Use generous letterspacing on display sizes (48px+)
- Use Space Mono for all numbers, stats, and technical data
- Keep body text 15–16px with 1.7 line height for readability
- Use Charcoal `#1A1A1A` for body text on light backgrounds

**DON'T:**

- Never write "Boon" or "BOON" — always "boon"
- Mix in other fonts — the three-font system is the entire toolkit
- Use light weights (300) for body text — readability suffers
- Use Crimson for long passages of body text — accents and highlights only

## 04 · Visual Language

Minimal, high-contrast, data-rich. Precision wellness — AG1's science-as-design approach combined with Harley Meds' bold palettes.

### Design Principles

| Principle | Description |
|-----------|-------------|
| **Light-first** | Cream backgrounds are the default canvas. Dark mode is reserved for packaging, heroes, and social. |
| **Crimson sparingly** | Red is a scalpel. Wordmark, one CTA per view, key data highlights. Never more than 10% of any composition. |
| **Science is design** | Turn purification data into beautiful visual content — diagrams, infographics, comparative charts. Show the science, don't just claim it. |
| **Ritual, not routine** | Photography and copy position water purification as a daily wellness ritual — not an appliance purchase. |
| **Generous space** | Let elements breathe. White space is a core brand asset, not empty real estate. |

### Photography Direction

| Aspect | Direction |
|--------|-----------|
| Style | Warm natural lighting, shallow depth of field, candid daily rituals — not posed studio shots |
| Subjects | Diverse, active people integrating boon into morning routines, kitchen scenes, workout recovery |
| Tone | Warm color grading, slight lift in shadows. Never cold/blue. Never overly filtered. |
| Product (light) | On cream/natural surfaces with soft directional lighting — editorial feel |
| Product (dark) | On dark surfaces with dramatic single-source lighting — packaging/social |

### Iconography

| Aspect | Spec |
|--------|------|
| Style | Geometric line icons, 1.5px stroke, rounded caps |
| Colors | Charcoal on light backgrounds, Cream on dark backgrounds |
| Grid | 24x24px base, 2px padding |
| Accent | Key icons can use Crimson fill sparingly — one per section max |

### Hard Avoids

- **NEVER** water droplet icons or logos — the most overused symbol in the category
- **NEVER** wave patterns, mountain/spring imagery — generic water cliches
- **NEVER** blue in any form — the brand deliberately rejects the water category default
- **NEVER** leaf, lotus, or tree motifs — exhausted "natural" signals
- **NEVER** stock wellness imagery (yoga poses, closed-eye serenity, overhead food styling)
- **NEVER** vague wellness language: "pure," "clean living," "your best self," "feel better"

## 06 · Voice & Copy Guidelines

How boon sounds across every touchpoint. Direct, confident, warm — never preachy.

### Tone Principles

| Trait | Description |
|-------|-------------|
| **Direct** | Short sentences. Active voice. Say it once, say it right. |
| **Honest** | Cite numbers. Name what we remove. Don't overclaim. "99.7% of contaminants" — not "the purest water on earth." |
| **Warm** | Confident, not cold. We talk like a smart friend, not a lab report. |
| **Bold** | We make claims that get attention. "Your water is lying to you." Then we back them up. |

### Copy Examples

**DO:**

- "99.7% of contaminants removed. Essential minerals kept."
- "Your morning ritual starts at the tap."
- "Six-stage purification. Zero plastic waste. One daily ritual."

**DON'T:**

- "Experience the purity of nature's finest water for your wellness journey."
- "Clean water for a clean life."
- "Our revolutionary filtration system leverages cutting-edge technology."

### Vocabulary

| Use | Avoid |
|-----|-------|
| Ritual | Routine |
| Purify / Purification | Filter / Filtration |
| Nourish | Hydrate (unless technical) |
| Vessel / System | Unit / Device / Appliance |
| Contaminants | Impurities / Bad stuff |
| Minerals preserved | Natural goodness |
| Transforms | Cleans / Fixes |
